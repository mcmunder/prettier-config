# @mcmunder/prettier-config

One prettier config to rule them all.

## Usage

Simply

```bash
npm install @mcmunder/prettier-config --save-dev
```

and point to the config in your `package.json`

```json
...
"prettier": "@mcmunder/prettier-config",
...
```

or `.prettierrc`:

```json
"@mcmunder/prettier-config"
```
